DOM の状態はリアクティブではなく、以下のようなことが必要になります：

- イベントコールバックを通じて状態の変化を監視する。
- API を通じてデータの変化を DOM の状態の変化に変換する。

axii は、よく使用される DOM 状態のリアクティブラッパーを提供しており、ユーザーは createStateFromRef を使用して DOM 状態のリアクティブラッパーをカスタマイズすることもできます。
既存のラッパーには以下のものがあります：
- dom size（DOM サイズ）
- position（位置）
- drag position（ドラッグ位置）
- scroll position（スクロール位置）
